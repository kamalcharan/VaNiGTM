Channel Orchestration
Agent behavior rules

The Orchestrator follows these rules when deciding which channel to use. Toggle to enable/disable.

P1
Email-first for cold outreach
Always start with email for first touch. Never WhatsApp or LinkedIn cold.
P1
WhatsApp requires opt-in
Only send WhatsApp after explicit opt-in or warm email reply.
P2
Channel escalation on no-response
If email gets no response after 3 attempts, try LinkedIn. If LinkedIn accepted, try WhatsApp.
P2
Cool-down between channels
Minimum 48 hours between touches on different channels for the same contact.
P3
Multi-channel for high-intent
Contacts with composite score > 80 get simultaneous email + LinkedIn + WhatsApp push.
P3
Weekend pause
No outbound messages on weekends. Queue for Monday morning delivery.
